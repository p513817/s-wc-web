# Prerequisite
1. Install `Python >= 3.10`
2. Install Python Module for `S-WC`
    ```bash
    python -m venv .env
    .\.env\Scripts\activate
    python -m pip install -r .\requirements.txt
    ```

# Usage
* Start :: Double Click the [`run.bat`](run.bat)
* Stop :: Double Click the [`stop.bat`](stop.bat)
